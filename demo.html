Accordion

A lightweight, CMS-injectable set of UI components managed via GitHub and served over jsDelivr CDN.
Create a folder under /components/my-component/ with matching CSS and JS files, add it to builds.json, and push to main.
Each bundle in builds.json defines which components to include and which dist folder to output to. GitHub Actions rebuilds all bundles on every push.
